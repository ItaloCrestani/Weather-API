<h1 align="center">🌤️ Weather App</h1>

Aplicação web simples para consultar o clima de qualquer cidade utilizando a API do OpenWeatherMap.

O projeto foi desenvolvido com HTML, CSS e JavaScript puro, consumindo dados de uma API externa e exibindo informações como temperatura, descrição do clima, umidade, vento e horário do pôr do sol.

## 🚀 Funcionalidades

- Buscar clima pelo nome da cidade
- Mostrar temperatura atual
- Mostrar descrição do clima
- Sensação térmica
- Umidade do ar
- Velocidade do vento
- Horário do pôr do sol
- Data local da cidade
- Bandeira do país
- Ícone representando o clima
